# dotnet-blazor-s3

Install dotnet

wget https://dot.net/v1/dotnet-install.sh
chmod +x dotnet-install.sh 
sudo ./dotnet-install.sh -c Current

Change `~/.bashrc`
```
export PATH="$PATH:$HOME/.rvm/bin:$HOME/.dotnet"
```
